# 性能优化



## 关闭透明度过渡

若游玩过程中出现负载过高的情况，可在元件库中将`对话框管理器`元件拖出，点击编辑节点图变量，将文字透明度过渡关闭，通常不需要关闭黑幕过渡，会导致视觉效果不好

拖出对话管理器元件，点击节点图的编辑节点图变量

![示例](/assets/dialogue-function-introduction/对话管理器.png)

里面有`是否开启文字透明度过渡`和`是否开启黑幕透明度过渡`

![示例](/assets/dialogue-function-introduction/对话管理器开关过渡.png)

## 延迟生成管理器实体

若关卡初始化负载过高导致无法试玩关卡，可以进入玩家身上的节点图变量中，延长管理器的生成时间，降低初始化的负担。

::: warning
由于管理器是进入游戏后才生成，因此不允许在玩家刚进入游戏后立马触发创建对话，如在玩家出生点创建碰撞箱触发对话等，需要等待管理器生成完毕才能触发对话
:::

在玩家模版这里，找到玩家身上的节点图`（勿删！）GameCoordinator`，点击编辑节点图变量

![示例](/assets/dialogue-function-introduction/玩家身上节点图.png)

里面有`对话管理器延迟生成`时间和`任务管理器延迟生成`时间，修改它们就可以让它们延迟生生

::: info
任务管理器是接在对话管理器生成之后的才开始自己的倒计时
:::


::: warning
2025.12.3以前下载的地图建议不要修改时间，目前发现已知Bug，将在2025.12.4凌晨修复上传
:::
