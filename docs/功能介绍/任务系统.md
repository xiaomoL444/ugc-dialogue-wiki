# 任务系统

其实这个并不算严格意义上的`系统`，我曾经做过的的任务结束自动接取关联任务，以及子任务的设计，现在全部被砍掉了，只留下了显示UI的部分，但是仅作为显示部分，其实就足够了

### 1.添加任务数据

前往`高级数据管理`，点击`任务配置列表`，我们在这里配置任务。

任务列表的整数代表人物的id，例如这里的1代表这个任务的id为1

::: warning
单个字典最大长度为`100`，不要超过这个长度，会导致无法进入游戏
:::

![示例](/assets/dialogue-function-introduction/任务配置列表高级数据管理.png)

点进结构体后，会看见几个关键字段，`标题`类似任务的章节，每次接取任务后会出现，并在2s后消失，留下`副标题`常驻。

调查预设点是填写预设点的点位，用来放在提醒玩家互动的位置（通常也表明任务位置或任务中心）

调查范围是当玩家处于这个范围内时，显示`已在任务范围内`，在范围外则显示距离任务中心点的距离，默认是-1

下面两个则是设置是否显示光标提示和特效提示光柱显示

### 2.接取任务

搜索`任务`，会有三个节点图

顾名思义，这里需要提醒的是，通常如果是一个任务接一个任务，直接使用`接取任务`就可以直接覆盖之前的任务并重新显示。

`完成任务`是让任务区域显示`任务完成`，`放弃任务`会直接关闭UI
