::: info `锁定对话`和`非锁定对话`
`锁定对话`：指在触发对话时，玩家会被固定在空气盒中，无法移动，并且关闭屏幕上基础的ui

`非锁定对话`：与锁定对话刚好相反，仅仅只显示文字，不做任何其他操作
:::

::: info `CG字幕文本`、`CG插画文本`和`中部文本`
本质上是同一种对话类型，只是文本在屏幕上的搞不不一样，并且中部文本为非锁定对话
:::
