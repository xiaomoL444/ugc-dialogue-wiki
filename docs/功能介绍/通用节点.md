# 通用节点

## 关闭UI

![示例](/assets/dialogue-function-introduction/关闭UI.png)

当奇匠通过`跳转对话`中的`跳转节点图事件`而脱离对话流时，需要自行使用`关闭对话UI`节点执行关闭对话的操作，通常情况下不需要强制关闭

## 剧情中限制位移的传送
（好绕的名字）

站桩对话、选项卡、黑幕文本、CG字幕文本、CG插画文本五种`锁定对话`，都会有空气盒子限制玩家移动。

因此如果在对话时，让玩家传送到某处，需要连带空气盒一起移动，因此这个节点的作用就是让玩家在对话传送时，连带空气盒和人物一起传送。

![示例](/assets/dialogue-function-introduction/剧情中限制位移的传送.png)

## 添加对话时需要隐藏的自定义UI

很多时候可能自己想要添加UI，但是站桩对话等`锁定对话`只会隐藏屏幕上的默认UI，因此需要该节点，添加自定义需要隐藏的UI。

::: info
想要隐藏的控件UI，只要设置一次即可，它会保存下来，并在每次进入`锁定对话`时把添加进的控件id进行隐藏，然后退出对话的时候再**打开**

所以请注意如果有想要关闭的UI请使用移除控件组，不然它还会在对话完后重新开启

:::

![示例](/assets/dialogue-function-introduction/对话隐藏UI.png)

## 创建调整选项卡比例选项

因千星的选项卡不支持动态文本，为保证奇匠的创作效率，这里使用了旁门左道实现了动态选项卡，缺点是不同宽度比例的设备，需要在游戏开始前，玩家手动调整UI比例，直接用该节点即可开启设置，并且如果一个关卡内有多个调整的地方，需要保证它们的`标识符`的字符串是独立唯一的，才能正确获取调整完毕事件

![示例](/assets/dialogue-function-introduction/创建调整选项卡比例选项.png)
